This site provides comprehensive resources for contributing to and maintaining CNCF projects.

**[Contributors](/docs/contributors/)** - Start your journey contributing to CNCF projects
- Getting started guides
- Project discovery
- Mentorship opportunities

### For Project Teams  
**[Projects](/docs/projects/)** - Resources for CNCF project maintainers and teams
- Project lifecycle guidance
- Governance frameworks
- Community building tools

### For Community Leaders
**[Community](/docs/community/)** - Coordination across the CNCF ecosystem  
- CNCF Staff resources
- TAG leadership guidance
- Cross-project initiatives

### For All Users
**[Resources](/docs/resources/)** - Tools, templates, and services
- CNCF project services
- Documentation templates
- Tool recommendations

## Quick Start

- **New to CNCF?** Start with [Getting Started](/docs/contributors/getting-started)

- **Looking for Templates?** Browse [Templates](/docs/projects/templates/)

This resource is maintained by the CNCF TAG Contributor Strategy to provide canonical guidance for the CNCF community. Content is organized around serving specific user needs and providing clear paths to success.
